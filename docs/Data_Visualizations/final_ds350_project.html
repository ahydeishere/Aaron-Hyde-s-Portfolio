Data Wrangling Issues/Solutions

(The data wrangling was mainly for the first data set. The second data set will remain the same because it is a map visual, not other data)

The main problems with the wrangling this data set is making it so that all the teams that were the same would still be classified. For example, There were a lot of teams that have moved to different locations over the last 2 decades, as well as a re branding of a team. The main teams that faced that issue were the Rams, Raiders, Chargers, and the Commanders. Because this data set has the name of the city or state first, some teams were called by the city then the mascot. So in order to change that, I set them to where their present day team are. The next hiccup was with the Washington Commanders. All the way until 2020, the commanders were known as the redskins. They changed their name to the Washington Football Team for a season or two, but then eventually landed on the Commanders as their team name. So in order to see how they did as a who over the last couple decades, I changed the name so that way it would be their present day names.

Another thing that I noticed with this data set is that there were a lot of different stats that were in this data set, but they were not important to me, so I filtered them out so that way they would not produce a problem for me in the long run. Some of these included mov (margin of victory), g(games played), and exp_pts_tot columns. These columns did not seem necessary to me while making these visuals, so I filtered them out here.

The other issue was that there were a lot of NA in the ties column. These NA were blank in the spreed sheet because there was no ties during that season. Other season had some ties. So I replaced all of the NA in the Ties column with 0 because there was a value.

season_map <- function(season) {

us_map <- map_data("state")

map_plot <- ggplot() +
  geom_polygon(data = us_map, aes(x = long, y = lat, group = group), 
               fill = "grey95", color = "white") +
  geom_point(data = nfl_geom_data %>% filter(year == {{season}}), 
             aes(x = long, y = lat, color = team), 
             size = 3, alpha = 0.7) +
  geom_text_repel(data = nfl_geom_data %>% filter(year == {{season}}), 
                  aes(x = long, y = lat, label = team),

  labs(title = paste0(season, " NFL Team Locations"),

  theme(legend.position = "none")

# Display the plot
  return(map_plot)

}

Yards per Season

yards_per_season <- function(season){
  
  season_yards <- nfl_data_cleaned %>%
    filter(year == season) %>%
    mutate(conference = str_extract(division, "AFC|NFC"))
  
  ggplot(season_yards, aes(x = total_yards, y = reorder(team, total_yards), fill = conference)) +
    geom_col() +
    theme_classic() +
    scale_x_continuous(labels = scales::comma) +
    scale_fill_manual(
      values = c("AFC" = "red2", "NFC" = "blue3")
    ) +
    theme(
      legend.position = "bottom"
    ) +
    labs(
      title = paste0(season, " Season"), 
      x = "Total Yards",
      y = "Team",
      fill = "Conference"
    )
}

Examples

In these visuals, you can see that there are different visuals for the total yards that each team was able to get over the different seasons. This is filtered out by the seasons, so you input what year you want to see and it will sort it from the team with the most years to the team with the least amount of yards. There is also a color filter so that way you can see which conference, either the AFC or the NFC teams, are at the top of the league during those seasons. For example, in the 2005 season 6 of the top 10 teams in total yards are in the AFC.

Point Differential Visual

point_differential_visualization <- function(team_name) {
  team_data <- nfl_data_cleaned %>%
    filter(team == {{team_name}}) %>%
    mutate(result = if_else(points_diff > 0, "Positive", "Negative"))

  ggplot(team_data, aes(x = year, y = points_diff, fill = result)) +
    geom_col() +
    geom_text(
      aes(label = points_diff,
          vjust = if_else(points_diff >= 0, -0.5, 1.5)),
      size = 3
    ) +
    scale_fill_manual(values = c("Positive" = "green4", "Negative" = "red3")) +
    theme_bw()+
    theme(legend.position = "none",
          panel.grid.major.x = element_blank(),
          panel.grid.minor.x = element_blank()) +
    labs(
      title = team_name,
      subtitle = "This is the point differential from 2003 to 2023",
      x = "Years",
      y = "Point Differential"
      )
  
}

Wins by Divisions

specific_division <- function(division_name) {
  
  team_specific <- nfl_data_cleaned %>%
    filter(division == division_name)       

  division_graph <- ggplot(team_specific, aes(y = wins, x = year, color = team)) +  
    geom_line() +
    scale_x_continuous(limits = c(NA, 2027), breaks = c(2000, 2005, 2010,2015,2020)) +
    scale_y_continuous(breaks = c(2,4,6,8,10,12,14,16))+
    scale_color_manual(values = nfl_colors) +
    theme_classic() +
    theme(
      legend.position = "none",
      plot.margin = margin(5, 90, 5, 5),
  axis.line.y = element_line(color = "black", linewidth = 0.5),
  axis.line.x = element_line(color = "black", linewidth = 0.5), 
  panel.grid.major.y = element_line(color = "gray80", linewidth = 0.3),
  panel.grid.major.x = element_line(color = "gray80", linewidth = 0.3),
  panel.grid.minor.y = element_line(color = "gray90", linewidth = 0.2)
) +
    geom_text_repel(
      data = team_specific %>% filter(year == max(year)), 
      aes(label = team),
      size = 3,
      nudge_x = 2,
      direction = "y",
      hjust = 0,
      segment.size = 0.3
    ) +
    coord_cartesian(clip = "off")+
  labs(
    title = division_name
  )
  
  return(division_graph)
}

Anaylsis:

In the graph above, I created a graph that would visualize the correlation between how many wins a team would get in a season with the amount of yards it would need. From the first visual, you can see that if a team is able to get more total yards, they would be able to win more games. The same is shown with the rushing and passing yards. Following making the scatter plot correlations, I made a visual of a box plot that would order the wins based the third quantile of total yards. This shows what win totals is most likely going to happen.
